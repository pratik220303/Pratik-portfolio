# Atelier – E-Commerce Frontend

Atelier is a modern, responsive **AI-inspired e-commerce frontend application** built using **React and Tailwind CSS**.  
It simulates intelligent shopping features like smart search, product recommendations, and dynamic product descriptions — all handled on the **frontend**.

---

##  Features

-  AI-inspired product recommendations (client-side logic)
-  Smart search with AI-style suggestions & trending searches
-  Cart management (add, remove, update quantity)
-  Persistent cart using localStorage
-  Dynamic AI-style product descriptions
-  Clean, premium UI with smooth animations
-  Fully responsive design

---

##  Tech Stack

- **Frontend:** React (JavaScript)
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Icons:** Lucide React
- **State Management:** React Hooks
- **Routing:** React Router
- **Storage:** Browser localStorage

>  This is a **frontend-only project** (no backend, no database).
